# TONMOVES

## Descrição do Projeto

TONMOVES é uma aplicação que permite criar vídeos que contam histórias a partir de arquivos de áudio. Utiliza tecnologias de IA para gerar imagens e vídeos, oferecendo uma experiência rica e interativa.

## Funcionalidades

- **Seletor de Fonte de Mídia**: Escolha entre gerar imagens com IA ou buscar fotos profissionais do Pexels.
- **Modos de Vídeo**: Selecione entre Slideshow ou Animações (Zoom/Pan).
- **Temas Visuais**: Personalize o visual do seu vídeo com temas como Cinematic, Neon, Nature, e mais.
- **Integração com APIs**: Suporte para provedores de imagem como Pollinations, Pexels, Together, e OpenAI.
- **Auto-Save**: Salva automaticamente o progresso do projeto.

## Configuração e Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/Tonx-Cloud/TONMOVES.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o aplicativo:
   ```bash
   npm run dev
   ```

## Uso

1. Faça upload de um arquivo de áudio.
2. Selecione o formato e modo do vídeo.
3. Escolha o tema visual e a fonte de mídia.
4. Clique em "Criar Vídeo" para gerar seu vídeo.
